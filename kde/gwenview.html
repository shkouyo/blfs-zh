gwenview-26.04.1

Introduction to Gwenview

Gwenview is a fast and easy-to-use image viewer for KDE.

Development versions of BLFS may not build or run some packages properly if LFS or dependencies have been updated since the most recent stable versions of the books.

Gwenview Dependencies

Required

Exiv2-0.28.8, kimageannotator-0.7.2, KDE Frameworks-6.26.0, and Little CMS-2.19.1

Recommended

libkdcraw-26.04.1

Optional

plasma-activities-6.6.5 and CFitsio

Installation of Gwenview

Install Gwenview by running the following commands: 

mkdir build &&
cd    build &&

cmake -D CMAKE_INSTALL_PREFIX=$KF6_PREFIX \
      -D CMAKE_BUILD_TYPE=Release         \
      -D BUILD_TESTING=OFF                \
      -W no-dev .. &&
make

This package does not come with a test suite.

Now, as the root user: 

make install
