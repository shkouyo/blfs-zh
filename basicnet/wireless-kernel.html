为无线网络配置 Linux 内核

在使用任何用户空间工具连接无线 AP 之前，必须配置 Linux 内核以正确驱动无线网卡。在内核配置中启用以下选项以及适用于您硬件的特定设备驱动程序，必要时重新编译内核： 

[*] Networking support --->                                                [NET]
  [*] Wireless --->                                                   [WIRELESS]
    <*/M>   cfg80211 - wireless configuration API                     [CFG80211]
    < /*/M>   Generic IEEE 802.11 Networking Stack (mac80211)         [MAC80211]

Device Drivers --->
  [*] Network device support --->                                   [NETDEVICES]
    [*] Wireless LAN --->                                                 [WLAN]

打开 “Wireless LAN” 子菜单，选择支持您硬件的选项。可以使用 pciutils-3.15.0 提供的 lspci 或 usbutils-019 提供的 lsusb 查看硬件配置。注意，无线网卡的许多选项（但并非全部）依赖于 CONFIG_MAC80211。加载正确的驱动程序后，接口将出现在 /sys/class/net 中，或显示在 ip link 命令的输出中。

许多无线网卡驱动程序需要固件。如果您已在内核配置中启用了正确的驱动程序，但它加载失败（出现类似 Direct firmware load for <filename> failed with error -2 的消息），这意味着您需要安装固件，否则无线网卡将无法工作。阅读 关于固件 了解更多详情。
