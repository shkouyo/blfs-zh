我应该阅读本书的哪个章节？

与 Linux From Scratch 不同，BLFS 并非设计为线性顺序阅读。LFS 提供了如何构建一个基础系统的说明，该系统可以演变为从 Web 服务器到多媒体桌面系统的任何形态。BLFS 试图引导您完成从基础系统到预期目标的整个过程，其中涉及大量选择。

每位阅读本书的读者都会想阅读某些章节。您当前正在阅读的 Introduction 包含通用信息。请特别注意 Chapter 2, Important Information 中的信息，其中包含有关如何解压软件、不同语言环境使用相关问题以及全书适用的各种其他注意事项。

大多数人接下来会想翻阅 Post LFS Configuration and Extra Software 部分。该部分不仅涉及配置，还涉及安全（Chapter 4, Security）、文件系统（Chapter 5, File Systems and Disk Management 和 UEFI 启动工具）、文本编辑器（Chapter 6, Text Editors）和 Shell（Chapter 7, Shells）。实际上，您可能希望在构建 LFS 系统时就参考本章的某些部分（尤其是文本编辑器和文件系统相关章节）。

在完成这些基础内容之后，大多数人至少会浏览本书的 General Libraries and Utilities 部分。该部分包含许多作为其他章节前提条件的信息，以及一些本身就很有用的项目（例如 Chapter 13, Programming）。您不必安装本部分中的所有库和软件包；每个 BLFS 安装步骤都会说明其依赖哪些其他软件包。您可以选择要安装的程序，然后查看它需要什么（别忘了检查嵌套依赖关系！）。

同样，大多数人可能还会查看 Networking 部分。该部分涉及使用 DHCP 和 PPP 等多种方法连接互联网或局域网（Chapter 14, Connecting to a Network），以及网络库（Chapter 17, Networking Libraries）和各种基本网络程序与工具。

处理好这些基础之后，您可能希望配置更高级的网络服务。这些内容在本书的 Servers 部分讨论。希望构建服务器的人可以在此找到良好的起点。请注意，该部分还包含多个数据库软件包的信息。

接下来的十二章涉及桌面系统。本书的这一部分以讨论 Graphical Components 的章节开始，该部分还涉及一些通用的基于 X 的库（Chapter 25, Graphical Environment Libraries）。之后，KDE、GNOME、Xfce 和 LXQt 各自拥有独立的部分，随后是 X Software。

本书随后继续讨论 Multimedia 软件包。请注意，许多人在开始 BLFS 之旅时可能就想使用本章中的 ALSA 说明；这些说明之所以放在这里，是因为这是最合乎逻辑的位置。

BLFS 主体部分的最后一部分涉及 Printing, Scanning and Typesetting。这对大多数桌面系统用户都很有用，即使是那些构建专用服务器的人也可能发现它很有价值。

我们希望您享受使用 BLFS 的过程。愿您实现构建完美个性化 Linux 系统的梦想！
