Python 模块

Python 模块简介

Python 模块包为 Python 语言添加了有用的对象。BLFS 中各包所使用的模块在此列出，连同它们的依赖关系。

重要

在 BLFS 中，我们通常使用 pip3 构建并安装 Python 3 模块。请注意，书中 pip3 install 命令应以 root 用户身份运行，除非用于 Python 虚拟环境。以非 root 用户身份运行 pip3 install 可能看起来工作正常，但会导致已安装的模块对其他用户不可访问。

pip3 install 默认不会重新安装已安装的模块。若要使用 pip3 install 命令升级模块（例如，从 meson-0.61.3 升级到 meson-0.62.0），请在命令行中插入 --upgrade。如果由于某种原因确实需要降级模块或重新安装相同版本，请在命令行中插入 --force-reinstall。

如果您之前安装过 make-ca-1.16.1 并已将文件从 /etc/profile.d 复制到新系统，则在安装本页任何 Python 模块之前，必须先安装 make-ca。
