组织结构

本书分为以下十四个部分。

第一部分 - 引言

这一部分包含理解本书其余内容所必需的基本信息。

第二部分 - LFS 的后续配置与额外软件

这里我们介绍基本配置和安全问题。我们还讨论了一系列在 LFS 主书中未涵盖的文本编辑器、文件系统和 shell。

第三部分 - 通用库与实用程序

在本节中，我们涵盖本书中经常需要的库以及系统实用工具。编程信息（包括重新编译 GCC 以支持其全部语言范围）为本部分作结。

第四部分 - 基本网络

这里我们解释当您不使用 LFS 主书中介绍的简单静态 IP 设置时，如何连接到网络。网络库和命令行网络工具也在这里涵盖。

第五部分 - 服务器

这里我们向您展示如何设置邮件和其他服务器（如 FTP、Apache 等）。

第六部分 - X 与窗口管理器

本部分解释如何设置基本的 X 窗口系统，以及一些通用的 X 库和窗口管理器。

第七部分 - KDE

本部分适用于希望使用 K 桌面环境（KDE） 或其部分组件的用户。

第八部分 - GNOME

GNOME 是桌面环境 KDE 的主要替代品。

第九部分 - Xfce

Xfce 是 GNOME 和 KDE 的轻量级替代品。

第十部分 - LXQt

LXQt 是 GNOME 和 KDE 的另一个轻量级替代品。

第十一部分 - 更多 X 软件

办公程序和图形网页浏览器对大多数人来说都很重要。它们以及一些通用的 X 软件，可以在本书的这一部分找到。

第十二部分 - 多媒体

这里我们涵盖多媒体库和驱动程序，以及一些音频、视频和 CD 刻录程序。

第十三部分 - 打印、扫描与排版（PST）

本部分涵盖文档处理，从 Ghostscript、CUPS 和 DocBook 等应用程序，一直到 texlive。
