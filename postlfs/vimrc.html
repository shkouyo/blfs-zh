/etc/vimrc 和 ~/.vimrc 文件

LFS 手册将 Vim 安装为其文本编辑器。此时应注意，存在 很多 不同的编辑应用程序，包括 Emacs、nano、Joe 等等。任何在网上（尤其是新闻组）待过一段时间的人，肯定至少见过一次争论，通常涉及 Vim 和 Emacs 用户！

LFS 手册创建了一个基本的 vimrc 文件。在本节中，我们将尝试增强此文件。启动时，vim 会读取全局配置文件（/etc/vimrc）以及用户特定的文件（~/.vimrc）。可以调整其中一个或两者，以适应您特定系统的需求。

以下是一个稍微扩展的 .vimrc，您可以将其放入 ~/.vimrc 中以提供用户特定的效果。当然，如果您将其放入 /etc/skel/.vimrc，则以后添加到系统的用户都将获得它。您也可以将文件从 /etc/skel/.vimrc 复制到系统上已有用户（如 root）的家目录中。如果您直接从 /etc/skel 复制任何内容，请务必设置权限、所有者和组。 

" Begin .vimrc

set columns=80
set wrapmargin=8
set ruler

" End .vimrc

请注意，注释标记是 " 而不是更常见的 # 或 //。这是正确的，vimrc 的语法略有不同。

以下是对此示例文件中每个选项含义的快速说明：

  • set columns=80：这只是设置屏幕上使用的列数。

  • set wrapmargin=8：这是从右窗口边框开始换行的字符数。

  • set ruler：这会使 vim 在屏幕右下角显示当前行号和列号。

关于 vim 众多 选项的更多信息，可以在 vim 内部阅读帮助。在 vim 中键入 :help 获取通用帮助，或键入 :help usr_toc.txt 查看用户手册目录。
